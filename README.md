# 🚀 Matplotlib Cheat Sheet

## Basic Plots

- plt.plot() – Line plot
- plt.scatter() – Scatter plot
- plt.bar() – Bar chart
- plt.barh() – Horizontal bar chart
- plt.hist() – Histogram
- plt.pie() – Pie chart
- plt.boxplot() – Box plot
- plt.stem() – Stem plot
- plt.step() – Step plot

## Labels

- plt.title() – Add title
- plt.xlabel() – X-axis label
- plt.ylabel() – Y-axis label
- plt.legend() – Show legend
- plt.grid() – Add grid

## Figure

- plt.figure() – Create figure
- plt.subplot() – Multiple plots
- plt.subplots() – Create subplots

## Axis

- plt.xlim() – Set X-axis limit
- plt.ylim() – Set Y-axis limit
- plt.xticks() – Set X ticks
- plt.yticks() – Set Y ticks

## Styling

- plt.colorbar() – Add color bar
- plt.annotate() – Add annotation
- plt.text() – Add text
- plt.fill_between() – Fill area

## Save & Display

- plt.show() – Display plot
- plt.savefig() – Save figure
- plt.close() – Close figure
